アプリケーション起動のコマンド

gunicorn -k flask_sockets.worker -b localhost:4000 -t 100 OHMIYA_APP:app

gunicornコマンドに関して
（ 参考：
  http://docs.gunicorn.org/en/stable/run.html
  http://docs.gunicorn.org/en/stable/settings.html#settings ）

-b BIND
バインドするサーバソケットを指定。
ローカルで起動する場合-> localhost:$(PORT)
社内ネットワーク全体で起動する場合-> 0.0.0.0:$(PORT)

-t INT
タイムアウトする秒数の指定

-k STR
使用するワーカの指定
